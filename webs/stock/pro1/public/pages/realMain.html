STEP 1

Download Daily NASDAQ 100 List

매일 nasdaq.com 선정 100대 리스트를 갱신합니다.

Download 100 List

STEP 2

Update Stock Data

NASDAQ 100 List 를 기반으로 당일의 종가 데이터를 DB 에 업로드 합니다. 이때 Yahoo API 를 사용하여 장 마감가, 거래량, 날짜데이터를 받아옵니다.

Update Daily Stock

STEP 3

Refresh All Stock Data

만약 NASDAQ 100 List 상에 변화가 생긴다면 10년치의 주가데이터를 모두 받아옵니다.

Restore 10 Years Stock

STEP 4

Run JAVA Program

nodeJS 의 child process module 을 이용하여 새로운 서브 프로세스를 실행시킵니다. K-Means 로 구현한 주가분석 JAVA 프로그램을 실행하게됩니다. 단, CPU 사용량이 많은 기계학습 알고리즘 이므로 여러번 실행시 서버가 다운될 수 있습니다.

Calc Daily Analisis

STEP 5

Make JSON FIle

Restful 한 서비스를 제공하기 위해 DB에 1회접근하여 모든 데이터를 JSON 형식으로 변환한 뒤 사용자 요청시 JSON 파일만 전송하여 DB 접근을 최소화 합니다. 주가분석 JAVA 프로그램의 결과값이 DB 에 업로드 된 후에 새롭게 분석된 데이터를 JSON 으로 변환합니다.

Update Daily Analisis JSON
